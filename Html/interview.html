Interview

Here's an interview with my friend Ola, a Software Development student, about her choice of this study and her experiences so far.

Why did you choose to study Software Development?

"I chose Software Development because I’ve always enjoyed working with computers, and I want to learn how to create my own applications and software."

What programming languages are you learning right now?

"I’m starting with HTML and CSS because it’s beginner-friendly, and I’m also learning a bit of JavaScript."

What was the first program or project you built?

"My first project was a Facebook page. It was basic, but it was fun to see it actually work!"

What do you find challenging about coding so far?

"Remembering all the syntax can be hard, and debugging my code when something goes wrong takes a lot of patience."

What’s one thing you enjoy about learning software development?

"I like how it feels like solving a puzzle, and I enjoy the sense of accomplishment when my code works."
